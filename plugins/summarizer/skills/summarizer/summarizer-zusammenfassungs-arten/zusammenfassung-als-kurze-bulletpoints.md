# Strategie: Kurze Bulletpoints

Geeignet für: Alle Arten von Dokumenten, wenn eine möglichst knappe Übersicht gewünscht ist.

## Namenskonvention für Ausgabedateien
Ausgabedateinamen vom Eingabedateinamen (ohne Erweiterung) ableiten:
- Ausgabe: `<Eingabedateiname_ohne_Erweiterung> - Bulletpoints.md`

Beispiel: Wenn die Eingabedatei `vortrag.pdf` ist, dann:
- Ausgabe → `vortrag - Bulletpoints.md`

## Aufgabe

Fasse die Inhalte so knapp wie möglich zusammen (Bulletpoints). Formatiere in Markdown. Bitte ohne bold / formatierte Überschriften, nur als hierarchische Bulletpoint-Liste.
